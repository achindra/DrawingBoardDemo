How to use the drawing board

  1. Select a drawing tool from the drop-down box: a line, a pencil, a rectangle or a circle;
  2. When you select either the line or the pencil tool you'll be able to pick a line width from a sub-menu. If you don't make a selection the line is se at a default of 2 pixels;
  3. When you select the Text tool you'll be able to write your text in the appropriate box, pick a font and a font size. Simply proceed to pick a color and click into the drawing board;
  4. Select a color from the drop-down box: black, red, blue, green, orange, yellow;
  5. Press the left-mouse button inside the canvas, keep it pressed while drawing. Release the mouse button each time you start on a new line or shape;
  6. Click the link at the top to erase all content from the canavas;
  7. Click the eraser to delete just bits of content from the canvas: just use the mouse the same way as the pencil tool. Make sure to click the deselect eraser tool before starting drawing again.

Your browser does not support HTML5 canvas element needed to use this application. Download Google Chrome or Firefox.
